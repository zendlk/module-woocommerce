== Description ==

Zend is telecommunication service provider based on Sri Lanka which you can use to send SMS,OTP,Flash messages to your customers. This plugin allows you to integrate Zend's SMS message gateway service into your WooCommerce store. 

= Features =

* Send SMS to your customers automatically for any WooCommerce order status.
* Receive SMS to one or multiple administrator/staff member's mobile number when your store got a new order.
* Customize the content of every automated SMS with custom parameter support.


== Installation ==

= Automatic Installation =

1. In the dashboard navigate to `Plugins > Add New`
2. Search for `Zend WooCommerce`
3. Locate the `Zend WooCommerce` by `Avanciro (Private) Limited` and click `Install Now`
4. After installation, click on `Activate Plugin` to activate

= Manual Installation =

1. Download the zip file using the `Download` button above in this page
2. In the dashboard navigate to `Plugins > Add New`
3. Click on `Upload Plugin`
4. Click `Choose File` then select the zip file you downloaded
5. Click `Install Now`, wait for the installation and click `Activate Plugin`

= Enabling the Landing Page =

1. In the dashboard, navigate to `WooCommerce > Settings`
2. Go to `Zend.lk` tab
3. Fill all options required, and save the settings


== Frequently Asked Questions ==

= How to sign-up for an account? =
Go to our website https://zend.lk and register for an account.

= Can this plugin send SMS? =
Yes, if you are registered Zend user, you can use this plugin to integrate with Zend and trigger automated SMS for WooCommerce order status updates and order note updates as well.
